# Mine Ronin

An onchain Minesweeper game built as a Base Mini App. It has classic difficulty
levels, custom boards, a 30-level Race mode, and a Weekly Challenge with a
leaderboard and an on-chain prize pool.

Players earn in-game coins and spend them in a shop on cosmetic skins that are
minted as NFTs on Base.

## Stack

Next.js, TypeScript, OnchainKit, wagmi, viem, Supabase, SIWE auth.
Contracts (Solidity / Foundry): skins NFT, in-game shop, weekly challenge pool.
Network: Base.
